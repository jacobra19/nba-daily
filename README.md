# nba-daily

## NodeJS CLI for getting the latest NBA games scores

<!-- local image -->
<img src="./demo.gif" alt="nba-daily" title="nba-daily" width="100%"/>

## Usage

```
npx nba-daily
// OR
npm install -g nba-daily
nba-daily
```

## Options

```--date```

you could specify the date you want to get the scores for.
The date should be in the format of ```DD/MM/YYYY```.
